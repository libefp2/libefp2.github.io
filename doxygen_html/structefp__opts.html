◆ terms

unsigned efp_opts::terms

Specifies which energy terms should be computed. This field is a collection of bit flags where each bit specifies whether the term is enabled (bit is set to 1) or disabled (bit is set to 0). To enable the term, use bitwise OR with the corresponding efp_term constant (e.g., terms |= EFP_TERM_ELEC). To disable the term, use bitwise AND NOT (e.g., terms &= ~EFP_TERM_POL).
