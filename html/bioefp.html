BioEFP is a scheme to split a macromolecule into fragments. Proteins can be split along C-Ca bonds, as shown below.

The model is described here. The original work used the following script step2.gen.gamess.pl Where is it?. For a new robust version of the script check out this BioEFP tutorial The tutorial is not there yet!

But wait, do I need to recompute fragment parameters for each new geometry of my polymer? Not necessary, if you did a homework. Check out this paper on Flexible EFP. A collection of scripts to help you set-up these jobs is Flexible EFP tutorial.
